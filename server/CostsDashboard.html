📊 Variación Cotizado vs Facturado Solo admin diferencia entre el costo cotizado al confirmar y la facturación real
Cotizado = lo que el área de viajes ingresa al confirmar costos (campo COSTO COTIZADO PARA VIAJE). Facturado = suma de TOTAL FACTURA + TOTAL FACTURA 2 + TOTAL FACTURA 3 (con fallbacks definidos en la configuración del dashboard). Solo se incluyen solicitudes con ambos valores > 0 (status RESERVADO o PROCESADO, sin OT cargada). Las solicitudes con OT válida no aplican porque se cargan directo a la OT.
